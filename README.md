# Anup Mazumdar — Portfolio

A fully interactive, editable personal portfolio website.

---

## 📁 Files in This Folder

```
portfolio/
├── index.html          ← Your entire portfolio (open this in a browser)
└── resume.docx         ← Your downloadable resume (keep this here!)
```

> ⚠️ **Important**: Keep `index.html` and `resume.docx` in the **same folder** so the download button works.

---

## 🚀 Deploy for FREE (Pick Any)

### Option 1 — Netlify Drop (Easiest, 30 seconds)
1. Go to **[netlify.com/drop](https://netlify.com/drop)**
2. Drag and drop your entire `portfolio` **folder** onto the page
3. Done! You get a live URL like `https://amazing-name-123.netlify.app`
4. (Optional) Go to Site Settings → Change site name to something like `anup-mazumdar`

### Option 2 — GitHub Pages (Free + Custom Domain)
1. Create a free account at [github.com](https://github.com)
2. Create a **new repository** named `anup-portfolio` (or your name)
3. Upload both files (`index.html` and `resume.docx`) to the repo
4. Go to **Settings → Pages → Branch: main → Save**
5. Your site goes live at `https://yourusername.github.io/anup-portfolio`

### Option 3 — Vercel (Fast CDN)
1. Go to [vercel.com](https://vercel.com) and sign up with GitHub
2. Click **Add New → Project**
3. Upload or connect your GitHub repo
4. Deploy — live in seconds at `https://anup-portfolio.vercel.app`

## ✏️ How to Edit Your Portfolio

### In the Browser (Easy)
1. Open `index.html` in Chrome/Firefox
2. Click the **✏ Edit** button (top-right corner)
3. You can now:
   - **Click any text** in the hero section to type new content
   - **Add items** using the `+ Add` buttons that appear at the bottom of each section
   - **Delete items** using the `×` buttons that appear on cards
4. Click **✓ Done** when finished
5. Your changes are **saved automatically** in the browser

> 💡 After editing in the browser, re-deploy to Netlify/GitHub to make changes public.

### Direct Code Edit (Advanced)
Open `index.html` in any text editor (VS Code, Notepad++) and find the `DEFAULT_DATA` object near the bottom — edit values there directly.
